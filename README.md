# genasset-web

Next.js **페이지 라우터** + React + Yarn 기본 세팅입니다.

## 시작하기

1) (사용자) Yarn 설치
2) 의존성 설치

```bash
yarn install
```

3) 개발 서버 실행

```bash
yarn dev
```

브라우저에서 `http://localhost:3000` 을 열면 됩니다.

## 스크립트

- `yarn dev`: 개발 서버
- `yarn build`: 프로덕션 빌드
- `yarn start`: 프로덕션 서버
- `yarn lint`: ESLint
